## 📄 7.2.1 - 设计工具清单.md

标签：#设计工具 #设计系统 #效率工具 #色彩规范 #图标资源

### 💻 四级子项：Figma/Sketch 插件：栅格预设、色彩对比检测、图标库

本清单收录了设计师日常工作中用于提升效率、确保规范性和获取资源的核心软件和插件。工具的选择旨在加速 **[[6.1.1 - 设计代币（Design Tokens）.md|设计代币]]** 和 **[[6.1.2 - 组件库设计.md|组件库]]** 的落地。

### 🛠️ 核心设计软件与插件

| 类型 | 软件/插件名称 | 核心功能 | 在设计规范中的应用 |
| :--- | :--- | :--- | :--- |
| **协作设计** | **Figma** / Sketch | **实时协作、组件化设计**。Figma 的变量功能可直接用于定义 **[[6.1.1 - 设计代币（Design Tokens）.md|设计代币]]**。 | 承载 **[[6.1.2 - 组件库设计.md|组件库]]** 和所有页面的主文件。 |
| **动效设计** | **Principle** / After Effects (AE) / Lottie | **高保真动效演示**。用于制作复杂的 **[[7.1.4 - 交互体验案例合集.md|微交互]]** 和 **[[5.2.2 - 页面转场动画.md|转场动画]]** 效果。 | 验证 **[[5.1.1 - 反馈动效设计.md|动效曲线]]** 和 **[[5.2.2 - 页面转场动画.md|时长]]**。 |
| **代码交付** | **Zeplin** / Figma Dev Mode | **设计稿标注与交付**。将设计稿中的属性自动转化为 CSS/代码参数，并引用 **[[6.1.1 - 设计代币（Design Tokens）.md|代币名称]]**。 | 确保交付代码遵循设计代币规范。 |
| **栅格预设** | **Layout Grids (Figma Plugin)** | 快速设置基于 **[[3.1.1 - 栅格系统规范.md|4 列或 8pt 栅格]]** 的布局结构。 | 保证所有界面布局的对齐和标准化。 |

| 工具名称 | 核心功能 | 在设计规范中的应用 | 关联规范 |

| **Coolors** / Adobe Color | **色彩方案生成与调和**。帮助设计师快速确定**[[2.1.1 - 品牌色规范.md|辅助色]]** 和**配色比例**。 | 辅助定义 **[[2.1.4 - 深色模式适配.md|深色模式]]** 下的色彩映射。 | [[2.1.1 - 品牌色规范.md]] |

* **色彩规范：** 色彩工具直接用于检验和生成 **[[2.1 - 移动端色彩体系.md|色彩体系]]**。
* **栅格系统：** 栅格预设工具是 **[[3.1.1 - 栅格系统规范.md|栅格系统]]** 的落地手段。
