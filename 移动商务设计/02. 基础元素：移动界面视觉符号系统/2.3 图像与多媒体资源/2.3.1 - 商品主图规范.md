## 📄 2.3.1 - 商品主图规范.md

标签：#图像系统 #商品卡片 #引流转化 #基础元素

### 🖼️ 四级子项：定义与核心逻辑：商品的第一视觉入口与转化驱动力

**商品主图规范（Product Main Image Specification）** 定义了商品在列表页、搜索结果页和详情页（PDP）首屏展示的核心图像标准。主图是用户接触商品的第一视觉要素，其质量和规范性直接决定了商品的**点击率（CTR）**和用户的**第一印象**。

**核心目标：**

1.  **高显著性（CTR）：** 在复杂的列表中实现 **[[1.1.1 - 视觉显著性]]**，吸引用户点击进入详情页。
2.  **信息传递效率：** 快速传达商品的核心卖点、材质和使用场景。
3.  **品牌统一性：** 确保主图风格符合平台和品牌自身的 **[[2.1.1 - 品牌色规范]]** 与调性。

### 📊 实操规范：主图的类型、尺寸与内容要求

电商主图通常需要提供多套不同用途的图片，以适应不同场景和营销策略。

| 主图类型 | 核心用途 | 尺寸与比例规范 | 内容要求与规范细节 |
| :--- | :--- | :--- | :--- |
| **标准白底图** | **列表页/搜索结果**：强调商品主体，去除干扰。 | **1:1 或 800x800 px**（高分辨率），需适配不同设备。 | **纯白背景（#FFFFFF）**，商品主体居中，清晰展示，无水印或牛皮癣（低端营销贴图）。 |
| **场景图/模特图** | **详情页首屏**：营造使用氛围，激发用户代入感。 | **3:4 或 9:16**（适配移动端竖屏），可结合 [[2.3.3 - 流媒体视觉规范.md|短视频]]。 | **聚焦核心卖点**，使用高品质摄影和光影，营造 **[[1.3.1 - 本能层设计]]** 的美感。 |
| **功能/细节图** | **详情页补充**：展示商品材质、局部细节、功能操作。 | 1:1 或 4:3 | 必须清晰展示材质纹理、接口、尺寸等细节，配合 [[2.2.3 - 信息层级排版法.md|辅助文字]] 标识。 |
| **营销图（可选）** | **活动引流**：在主图上添加限时促销信息。 | 1:1 或 3:4 | 必须遵守平台规范，**促销标签**（如“限时折扣”）应靠边角放置，且使用 [[2.1.2 - 营销色与功能色.md|营销色]] 突出，但不能遮挡商品主体。 |

### 🔍 关键设计要素：如何在列表中脱颖而出

1.  **主体突出原则：** 在白底图中，商品主体应占据画面的 **80% 以上**空间，确保在小尺寸列表格中也能清晰识别。
2.  **视觉差异化：** 即使是白底图，也需要利用**商品摆放角度、光影细节**来创造差异。例如，服装类商品可以通过独特的模特姿势或局部特写，实现 **[[1.1.2 - 视线流设计]]** 的引导。
3.  **水印与贴图规范：** 平台 Logo 或 **[[1.2.2 - 信任背书设计]]** 标识（如“官方认证”）应以**低透明度、简洁的图标**形式放置在不干扰商品主体的角落，避免分散用户的注意力。
4.  **色彩饱和度：** 主图的色彩饱和度应略高于环境图，利用 **[[1.1.1 - 视觉显著性]]** 吸引目光，但必须避免过度修图导致的失真。
