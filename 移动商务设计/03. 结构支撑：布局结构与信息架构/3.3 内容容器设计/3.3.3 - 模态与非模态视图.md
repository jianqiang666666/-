## 📄 3.3.3 - 模态与非模态视图.md

标签：#布局结构 #交互视图 #用户体验 #转化层设计 #UI美化

### 🧱 四级子项：定义与核心逻辑：打断程度与用户注意力的控制

**模态与非模态视图（Modal and Non-Modal Views）** 定义了界面中信息展示或交互流的两种基本方式，它们的核心区别在于**对用户当前任务的打断程度**和**对用户注意力的独占性**。

**核心目标：**

1.  **任务聚焦（模态）：** 用于强制用户在继续主任务前，必须完成当前操作或确认重要信息，常用于 **[[4.3.2 - 结算页设计.md|转化流程]]**。设计上必须通过**视觉隔离**（如遮罩）来独占用户注意力。
2.  **辅助提示（非模态）：** 用于提供短暂的、非关键的系统反馈或次要信息，不阻碍用户进行主任务。设计上追求**克制**和**快速消失**。

---

### 🚨 模态视图（Modal Views）规范：强打断与视觉隔离

模态视图会**冻结或遮罩**主界面，用户必须与其交互才能解除。其美化重点在于**转场动效、背景遮罩的质感**和**内容容器的精细度**。

| 模态类型 | 触发与解除 | 视觉表现与规范细节（UI/美化） | 电商应用场景举例 |
| :--- | :--- | :--- | :--- |
| **全屏模态（Full Screen Modal）** | 用于复杂且多步骤的临时流程。必须提供 **返回/关闭** 按钮。 | **转场美化：** 采用 **[[3.2.3 - 页面间的跳转与转场.md|从底部向上覆盖]]** 的平滑转场动效。背景为实色，提供一个完整的独立工作区，**顶部导航栏**设计应清晰且专注于任务。 | 登录/注册流程、创建收货地址、复杂筛选面板。 |
| **底部操作表（Action Sheet/Bottom Sheet）** | 用于提供与当前上下文紧密相关的**有限列表选项**。 | **容器美化：** 底部容器应具有与界面 **[[3.3.1 - 卡片式设计（Card UI）.md|卡片一致]]** 的 **圆角（Corner Radius）**。背景使用 **深色半透明遮罩**（如 `#000000` 60% 透明度）来优雅地模糊主界面。 | 分享选项、快速选择商品 SKU、优惠券选择器。 |
| **对话框（Alert/Dialog）** | 用于**强制性确认**或**重要警告**。通常只包含**是/否**或**确认**按钮。 | **视觉焦点：** 位于屏幕中央，周围有**深色半透明遮罩**。内容区应使用 **[[3.3.1 - 卡片式设计（Card UI）.md|柔和阴影]]**，按钮遵循 [[2.1.2 - 营销色与功能色.md|功能色]]（如**警告红 🔴**）以强化语义。 | 退出登录确认、订单取消警告、系统权限申请。 |

### 💬 非模态视图（Non-Modal Views）规范：弱打断与克制反馈

非模态视图在不中断用户当前任务的情况下提供信息，用户可以忽略。其美化重点在于**动画的流畅性、简洁的视觉**和**信息层级的克制**。

| 非模态类型 | 核心用途 | 视觉表现与规范细节（UI/美化） | 电商应用场景举例 |
| :--- | :--- | :--- | :--- |
| **Toast 提示** | **系统反馈**：提供短暂的操作成功或失败信息。 | **简洁设计：** 自动消失，不含按钮。位于屏幕**底部中央**，使用 **中性色背景** 和 **[[2.2.3 - 信息层级排版法.md|Caption 级别字号]]**。动效为 **快速淡入淡出**（< 200ms）。 | “已加入购物车”、“网络连接失败”的瞬时反馈。 |
| **Snackbar** | **可操作提示**：提供短暂信息，且包含一个可逆转的操作按钮。 | **流畅动效：** 从底部滑入并自动消失，持续时间略长。**按钮**使用 **[[2.1.1 - 品牌色规范.md|品牌色]]** 高亮。应提供 **[[5.1.2 - 状态变更反馈.md|撤销操作]]**，文案简洁且富有引导性。 | “已删除该商品 🗑️ [撤销]”的提示。 |
| **横幅/浮层 (Banner)** | **信息/营销提示**：持续显示的重要通知或优惠信息。 | **营销美化：** 位于顶部导航栏下方或底部，使用 **[[1.2.1 - 紧迫感构建.md|营销色]]**，并配以 **[[1.1.1 - 视觉显著性.md|高对比度文案]]**。通常带有可关闭按钮 **❌**。 | 促销活动提醒、App 升级/实名认证提示。 |

> ****
> *图示：典型的对话框模态视图，通过**深色遮罩**实现视觉隔离，**圆角和阴影**提升 UI 质感，**红色按钮**强化确认操作的语义。
